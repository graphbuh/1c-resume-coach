# 1C Resume Coach

AI-система автоматического анализа резюме специалистов 1С и формирования рекомендаций по карьерному развитию на основе анализа рынка вакансий.

Проект разработан в рамках хакатона и учебной практики.  
Система анализирует резюме, извлекает навыки, определяет профессиональную роль кандидата и сравнивает профиль кандидата с реальными требованиями рынка.

---

# Основные возможности

Система позволяет:

• автоматически извлекать навыки из текста резюме  
• классифицировать профессиональную роль кандидата  
• сравнивать резюме с профилем рынка вакансий  
• выявлять недостающие навыки  
• формировать рекомендации по улучшению резюме  
• предлагать направления профессионального развития  

---

# Архитектура решения

Pipeline системы состоит из нескольких этапов:

1. Сбор и очистка данных о вакансиях  
2. Удаление дублей вакансий  
3. Формирование словаря навыков  
4. Извлечение навыков из вакансий  
5. Классификация ролей вакансий  
6. Обучение нейросетевой модели (baseline)  
7. Анализ резюме кандидата  
8. Gap-анализ резюме относительно рынка  
9. Генерация рекомендаций  
10. Web-интерфейс (Streamlit)

# Запуск системы

1. Поместить резюме в папку
data/resumes_txt

в формате **.txt**

---

2. Запустить интерфейс
streamlit run scripts/09_streamlit_app.py

---

3. Открыть браузер
http://localhost:8501



---

# Как работает анализ

После загрузки резюме система:

1. извлекает навыки  
2. определяет роль кандидата  
3. сравнивает профиль с рынком вакансий  
4. рассчитывает coverage навыков  
5. показывает недостающие навыки  
6. формирует рекомендации
